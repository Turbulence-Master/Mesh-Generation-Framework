A free F\-O\-R\-T\-R\-A\-N unit number is needed in order to open a file with the O\-P\-E\-N command. If I\-U\-N\-I\-T = 0, then no free F\-O\-R\-T\-R\-A\-N unit could be found, although all 99 units were checked (except for units 5, 6 and 9, which are commonly reserved for console I/\-O).
